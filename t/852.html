  1. 端点星是基于 jekyll，在 github 上搭建的。你用关键词 “GitHub page 博客” 能搜到一堆教程。端点星本身也是开源的 https://github.com/Terminus2049/Terminus2049.github.io

  2. 被删文章有两个来源

  • 主动式：将你认为可能会被和谐的文章 url 发送到 archive.is，可以使用telegram机器人 https://t.me/Archive2049Bot
  • 被动式：港中文的 Wechatscope 项目会跟踪微信公众号，检测是否被删，并将被删文章以 api 的形式提供给外界使用，端点星基于它的 api 做了前端展示： http://206.189.252.32:3838/Wechatscope/
