辣個十里山路不換肩的男人

《習近平回憶插隊歲月 ：扛２００斤麥子，十里山路不換肩》當中紀錄了習近平的一段自述：『我幾乎那一年３６５天沒有歇着，除了生病。下雨刮風我在窯洞里，跟他們鍘草，晚上跟着看牲口，然後跟他們去放羊，什麼活都干，因為我那個時候扛２００斤麥子，十里山路不換肩。』

習近平此說法被各大媒體挖掘出來後，網友認為那是習近平誇大其詞的吹牛，而成了大家冷嘲熱諷的話題。我們同樣也感到好奇，所以我們收集了數據，要用數學和科學來探討究竟有沒有可能”扛２００斤麥子走十里山路不換肩”。

我們將此專題分成以下步驟：

(一)收集習近平的年齡、身高、體重，並找出黃土高原的山路剖面圖

(二)記錄下習近平爬山時的高度、距離

(三)利用物理學求出他作的功、位能、消耗的能量、卡路里

(四)用生物學的角度探討

(五)探討可能性

(1)

經過我們的查詢， 1969年 1月，因文化大革命對家庭帶來的衝擊而被迫中止學業，作為知青*前往陝西省延安市川縣梁家河村參加勞動與工作。（註：知識青年，簡稱知青，是中華人民共和國對知識分子的歷史名詞）

而習近平現年 66歲，經由下列算式:

2020 - 1969 = 51

66 - 51 = 15

計算出習近平當時應該是十五歲。

我們運用了Google Map查出了梁家河村的位置

經查詢，梁家河在黃土高原附近，所以證實了習近平搬麥子走山路言不虛，

於是我們便開始證明這件事的合理性。

所以用Google Earth查出了數據

十里轉換成國際單位制為五公里，兩百斤經換算120公斤

(三)

我們推估他當時為 70 公斤

由位能公式我們可以得知他所消耗的量

(因往下走所消耗的能量等同於平路，以不列入計算)

我們得出此公式：

(體重 +麥子重量 )* 重力加速度 *( 上升走的距離 )

(120+70)9.8 (110+135+47+19+18+12+52+14)=757834 焦耳

由作功公式我們可以得知他所消耗的能量

作功公式為：

(體重 +麥子重量 )* 重力加速度 *( 全程的距離 )

(120+70)9.8(5*1000)=9310000焦耳

所以他消耗的能量為上升的能量 (位能 )+ 往前走的能量 (作功 )

也就是 :

757834+9310000=10067834焦耳

一大卡=4186焦耳

10067834/4186≌2405大卡

所以消耗的熱量為 2405 大卡

所以消耗的熱量相當於 :

(總熱量 )/(一份食物的熱量) =(需要幾份食物 )

2405 /530 ≌ 4.5 個大麥克漢堡

2405 /470 ≌ 5 份大薯

(4)

真是太不可思議了，竟然只要五份大薯的能量就可以扛 120 公斤的麥子走五公里，我們對這個結果感到懷疑，因此我們去尋找了更多的資料，最後好不容易找到了這個網站。

並以人:70 公斤爬山的時間:60分鐘的前提，繪製出以下表格

平均每公斤的消耗 (扣掉體重 ) = 消耗的卡路里 / (70+ 背負的重量 )

為了要求出背負 120 公斤所消耗的卡路里，我們將平均每公斤的消耗標點

畫出以下表格

如上圖所示，我們的標點並非以線性成長，於是我們用近似線求出方程式

y = 平均每公斤的消耗 (扣掉體重 )

x = 背負的重量

y = 0.0002x^3^- 0.0108x^2^ + 0.2822x +4.1224

並用以上方程式回推出背負120公斤時，平均每公斤的消耗為:

0.0002* 120^3^ - 0.0108* 120^2^ + 0.2822* 120 + 4.1224= 228.0664

然後再將

228.0664* (120+ 70) = 43332 .6

算出其消耗的卡路里為43332.6大卡

相當於 43332/344 = 125 個麥香魚堡

跟我們原本算出的 2405 大卡天差地遠，但到底為什麼會這樣呢?

於是我們又去查詢了更多的資料，找到了兩個原因:

1:人的功率轉換並不是100%

2:物理學的做功只有考慮到平行的做功，並沒有考慮到扛東西走的做功

(5)

雖然算出了他會消耗的熱量，但我們還有許多覺得不合理的地方，所以我們查詢了一些國內外網站的資料，得知一般有訓練過的人(70公斤)在健行時背負在健行時背負的重量約為其體重的30%\ 70*0.3 = 21公斤

而我們的主角背負公斤數為 120 公斤，比一般登山客多了快五倍，有研究指出，當重量超過30%時，便會影響移動速度，尤其是在不對稱的情況下，更會大幅增加跌倒的機率。
