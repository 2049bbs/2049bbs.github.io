EntryNodes // 指定入口节点，Guand节点

MiddleNodes // 指定中间节点

ExitNodes // 指定离开节点

ExcludeNodes // 排除节点，用来排除蜜罐节点，<del>我相信不存在这种节点</del>

ExcludeExitNodes //排除出口节点

配置方法：

修改torrc文件，增加对应的配置 例如,强制排除中国的节点

    ExcludeNodes            {cn}   StrictNodes 1
