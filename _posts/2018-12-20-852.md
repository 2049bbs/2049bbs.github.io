            *   主动式：将你认为可能会被和谐的文章 url 发送到 archive.is，可以使用telegram机器人
            https://t.me/Archive2049Bot

            *   被动式：港中文的 Wechatscope 项目会跟踪微信公众号，检测是否被删，并将被删文章以 api
            的形式提供给外界使用，端点星基于它的 api 做了前端展示：
            http://206.189.252.32:3838/Wechatscope/
